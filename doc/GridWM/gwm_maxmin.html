%function [row_col LinearIndex]=gwm_maxmin(data,num_of_values,extremum_type)
   Select number of NUM_OF_VALUES MAX/MIN values from matrix and return
   its indexes in [row col] notation or in linear index (Linear index can 
   be processed with ind2sub)
       INPUT:
               data            - matrix for searching MIN/MAX values
               num_of_values   - number of extreme values to return
               extreme_type    - which extreme type to use ('max' or 'min')
       OUTPUT:
               row_col         - matrix where in 1st column we store row
                                 index and in 3nd column - column index
                                        row  column
                                        [1     2 
                                         2     3
                                         4     3]
               LinearIndex     - we store here the linear index which
                                 further can be processed with ind2sub function
