rle

PURPOSE

data = rle(x) (de)compresses the data with the RLE-Algorithm

SYNOPSIS

function data = rle(x)

DESCRIPTION 

 data = rle(x) (de)compresses the data with the RLE-Algorithm
   Compression:
      if x is a numbervector data{1} contains the values
      and data{2} contains the run lenths

   Decompression:
      if x is a cell array, data contains the uncompressed values
